# Sternik Motorowodny — приложение для подготовки

PWA приложение с 176 вопросами для подготовки к экзамену. Работает как обычный сайт **и** устанавливается на Android как полноценное приложение.

## Что внутри

- **176 вопросов** на польском и русском (с переключателем PL / RU)
- **Режим изучения** — все вопросы по порядку, правильный ответ подсвечен зелёным
- **Режим теста** — вопросы вперемешку, выбираешь ответ → сразу видишь правильно/неправильно, показывается верный, через секунду переходим дальше
- Прогресс изучения сохраняется (можно закрыть и продолжить с того же места)
- Лучший результат и серия правильных ответов сохраняются
- Работает **офлайн** после первой загрузки
- Тёмная тема автоматически по настройкам системы
- Поддержка клавиатуры: 1/2/3 или A/B/C — выбор ответа, ←/→ — переключение

## Как запустить локально (для теста)

В папке `sternik-app`:

```bash
python3 -m http.server 8000
```

Открыть в браузере: http://localhost:8000

## Как установить на Android (как настоящее приложение)

Самый простой способ — захостить файлы и открыть на телефоне:

### Вариант 1 — GitHub Pages (бесплатно, 5 минут)

1. Создать репозиторий на github.com, загрузить туда содержимое папки `sternik-app`
2. Settings → Pages → Source: deploy from branch `main`, folder `/`
3. Подождать пару минут, открыть полученный URL на Android в Chrome
4. В меню Chrome (три точки) → **«Установить приложение»** или **«Добавить на главный экран»**

После установки иконка появится на рабочем столе, открывается без браузерных рамок — выглядит и ведёт себя как нативное Android приложение.

### Вариант 2 — Netlify / Vercel / Cloudflare Pages

Перетащить папку `sternik-app` на netlify.com → drop → готово, ссылка готова через 30 секунд.

### Вариант 3 — Локально на телефоне без хостинга

Можно использовать приложение типа `Termux` + `python3 -m http.server` и открыть на телефоне через `localhost`. Но проще через хостинг.

## Структура файлов

```
sternik-app/
├── index.html              UI
├── styles.css              Стили (с тёмной темой)
├── app.js                  Вся логика приложения
├── questions.json          176 вопросов с переводами
├── manifest.json           PWA-манифест
├── sw.js                   Service worker (офлайн режим)
├── icon.svg                Иконка (векторная)
├── icon-192.png            Иконка для Android
├── icon-512.png            Иконка для Android (большая)
├── icon-maskable-512.png   Иконка с safe zone (для адаптивных иконок)
└── favicon.png             Favicon для браузера
```

## Если нужно превратить в настоящий APK

Сервис **PWABuilder** (https://www.pwabuilder.com/) умеет автоматически собирать APK из PWA. Просто вставить URL хостинга и скачать готовый APK — без Android Studio.
